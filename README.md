# Network Topology

- Визуализатор топологии сети на React + TypeScript с использованием Cytoscape.
- Загружает структуру (узлы и связи) с бэкенда, фильтрует висящие связи, сохраняет позиции узлов в localStorage и применяет fcose/preset layout.

Требования
- Node.js (рекомендуется LTS)
- pnpm (или используйте npm/yarn, но команды в примерах — для pnpm)

Быстрый старт
1. Установите зависимости:

```bash
pnpm install
```

2. Запустить дев-сервер для разработки:

```bash
pnpm dev
```

3. Сборка для продакшена:

```bash
pnpm build
pnpm preview
```

Переменные окружения
- В разработке базовый URL API задаётся через `VITE_API_BASE_URL`. По умолчанию проект использует прокси `/api` (см. `vite.config.ts`). Для локальной разработки создайте файл `.env.development` и при необходимости переопределите:

```env
VITE_API_BASE_URL=/api
```
